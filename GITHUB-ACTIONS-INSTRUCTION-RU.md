# Сборка HotelChat APK через GitHub Actions

Эта версия уже содержит файл:

```text
.github/workflows/build-apk.yml
```

GitHub автоматически:

1. Загружает проект.
2. Устанавливает Java 17.
3. Загружает стабильную версию Flutter из официального репозитория.
4. Создаёт стандартные Android-файлы.
5. Загружает зависимости.
6. Проверяет исходный код.
7. Собирает универсальный APK.
8. Публикует APK в разделе Artifacts.

## Важно

APK из этого workflow предназначен для ручной установки сотрудникам гостиницы.
Для публикации в Google Play потребуется отдельный постоянный ключ подписи.

## Как загрузить проект

1. Создайте новый репозиторий GitHub.
2. Можно выбрать Private, чтобы исходники не были публичными.
3. Распакуйте архив на компьютере.
4. Откройте созданный репозиторий.
5. Нажмите Add file → Upload files.
6. Перетащите содержимое папки проекта, включая скрытую папку `.github`.
7. Нажмите Commit changes.

## Как запустить сборку

После первой загрузки сборка запустится автоматически.

Для ручного запуска:

1. Откройте вкладку Actions.
2. Выберите Build HotelChat Android APK.
3. Нажмите Run workflow.
4. Выберите main.
5. Ещё раз нажмите Run workflow.

## Как скачать APK

1. Откройте завершённый запуск с зелёной галочкой.
2. Прокрутите страницу до раздела Artifacts.
3. Нажмите HotelChat-Mobile-APK-N.
4. Скачается ZIP.
5. Распакуйте его.
6. Внутри будет APK и файл SHA-256.

## Если сборка красная

Откройте неуспешный запуск и нажмите на красный этап.
Скопируйте его полный текст ошибки.

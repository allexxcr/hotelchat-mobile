# HotelChat Mobile 1.1

Android-приложение сотрудников с Firebase Cloud Messaging.

## Firebase

- Project ID: `hotelchat-e9c5f`
- Android package: `online.ognispb.hotelchat`
- Конфигурация: `firebase/google-services.json`

Закрытый service-account JSON в этот проект не входит и в GitHub
загружаться не должен.

## Новое в версии 1.1

- push-уведомления при новых сообщениях гостя;
- запрос разрешения уведомлений на Android 13+;
- открытие конкретного обращения по нажатию уведомления;
- автоматическая регистрация и обновление FCM-токена;
- обновление списка чатов при foreground-сообщении;
- выбор фото из галереи или камеры;
- предпросмотр вложения и удаление до отправки;
- текст и фотография отправляются одной кнопкой;
- полный журнал сборки в GitHub Artifacts.

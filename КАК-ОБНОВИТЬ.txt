1. Распакуйте архив.
2. В GitHub откройте Code → Add file → Upload files.
3. Загрузите всё содержимое папки поверх существующих файлов.
4. Нажмите Commit changes.
5. Откройте Actions → Build HotelChat Android APK.
6. После зелёной галочки скачайте HotelChat-Mobile-APK-N.
7. При красной ошибке скачайте Artifact HotelChat-Build-Log-N и передайте файл build-apk.log.
